# AtCoderScraper

## Example Usage

```bash
mkdir AtCoderCodes
cd AtCoderCodes
git init
git submodule add https://github.com/NULLCT/AtCoderScraper
pip install -r ./AtCoderScraper/requirements.txt
python ./AtCoderScraper/src/scraper.py     # then enter atcoder username
```
and push this repo to somewhere
